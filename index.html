Introduction

EZCode is a high-level scripting language designed to be simple and easy to understand...

File Format

File Extension: .ez
Example File: game.ez

Basic Syntax

Statements and Commands

create object at 100, 200
move object right by 10

Variables

set player_health to 100
set player_name to "Hero"

Functions

function say_hello
    print "Hello, World!"
end function

call say_hello

Comments

// This is a comment
set player_health to 100 // Setting initial health

Control Flow

If-Else Conditions

if player_health <= 0
    print "Game Over"
else
    print "You are still alive"
end if
