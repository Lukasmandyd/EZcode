EZCode Full Guide

Installation

1. Download the latest version from here.

2. Run the installer and follow the steps.

3. Open a terminal and type EZ myprogram.ez to run your first script!

Basic Syntax

Variables

set x to 10

Output

write "Hello, EZCode!"

Input

ask for "What is your name?" as username

Conditions

if x is greater than 5
    write "X is big!"
end if

Loops

repeat 5 times
    write "Hello!"
end repeat
