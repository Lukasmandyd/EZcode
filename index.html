Introduction

EZCode is a high-level scripting language designed to be as simple and easy to understand as possible. It aims to make programming accessible to everyone, with an intuitive syntax that is readable and straightforward. This makes it perfect for beginners and non-programmers, while still being powerful enough to create complex applications, games, and more.

File Format

File Extension: .ez
Example File: game.ez

EZCode files are plain text files that can be opened and edited with any text editor, such as Notepad or Visual Studio Code.

Basic Syntax

Statements and Commands

create object at 100, 200
move object right by 10

Variables

set player_health to 100
set player_name to "Hero"

Functions

function say_hello
    print "Hello, World!"
end function

call say_hello

Comments

// This is a comment
set player_health to 100 // Setting initial health

Control Flow

If-Else Conditions

if player_health <= 0
    print "Game Over"
else
    print "You are still alive"
end if

Loops

For Loop:

for i from 1 to 10
    print i
end for

Repeat Forever Loop:

repeat forever
    move player up by 5
end repeat

Basic Game Mechanics

Creating Objects

create player at 100, 150
create enemy at 200, 150

Moving Objects

move player up by 5
move enemy left by 5

Interaction Between Objects

if player collides with enemy
    set player_health to player_health - 10
end if

Printing Messages

print "Game Started!"

Advanced Features

Functions and Parameters

function add_numbers with a, b
    return a + b
end function

set result to call add_numbers with 5, 10
print result

Arrays (Lists)

set scores to [10, 20, 30, 40]
set first_score to scores[0]

Random Numbers

set random_number to random between 1 and 100

Error Handling

try
    set result to divide 10 by 0
catch error
    print "Error: Division by zero"
end try

Game Engine Integration

Creating and Moving Objects

create sprite "player" at 100, 100
move sprite "player" left by 5

Playing Sounds

play sound "explosion.wav"

Checking Collisions

if sprite "player" collides with sprite "enemy"
    print "Game Over!"
end if

Saving and Loading Data

Saving Data

save data "player_data.txt" as player_name, player_health

Loading Data

load data "player_data.txt" into player_name, player_health

Getting Started

Download EZCode interpreter or compiler from the official site.
Create a new .ez file and open it with any text editor.
Write your first program, such as printing "Hello, World!", and run it!
