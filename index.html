EZCode Language Documentation

1. Introduction

EZCode is a high-level scripting language designed to be as simple and easy to understand as possible. It aims to make programming accessible to everyone, with an intuitive syntax that is readable and straightforward. This makes it perfect for beginners and non-programmers, while still being powerful enough to create complex applications, games, and more.

2. File Format

File Extension: .ez

Example File: game.ez

EZCode files are plain text files that can be opened and edited with any text editor, such as Notepad or Visual Studio Code.

3. Basic Syntax

EZCode uses a natural language syntax, which means it tries to read like English. Below is a list of basic syntax elements:

3.1 Statements and Commands

All commands are written in plain English. No semicolons or complex punctuation are needed (just simple line breaks between statements).

create object at 100, 200
move object right by 10

3.2 Variables

EZCode allows the use of variables to store and manipulate data.

set player_health to 100
set player_name to "Hero"

3.3 Functions

EZCode supports defining functions that allow for code reuse.

function say_hello
    print "Hello, World!"
end function

call say_hello

3.4 Comments

To make your code more readable, you can add comments. Comments are ignored by the interpreter and are for human readers only.

// This is a comment
set player_health to 100 // Setting initial health

4. Control Flow

EZCode allows for basic control flow constructs like if-else conditions, loops, and repeats.

4.1 If-Else Conditions

if player_health <= 0
    print "Game Over"
else
    print "You are still alive"
end if

4.2 Loops

For Loop:

for i from 1 to 10
    print i
end for

Repeat Forever Loop:

repeat forever
    move player up by 5
end repeat

5. Basic Game Mechanics

You can build games using EZCode by creating objects (like players, enemies, and items) and controlling them through simple commands.

5.1 Creating Objects

create player at 100, 150
create enemy at 200, 150

5.2 Moving Objects

move player up by 5
move enemy left by 5

5.3 Interaction Between Objects

if player collides with enemy
    set player_health to player_health - 10
end if

5.4 Printing Messages

print "Game Started!"

6. Advanced Features

6.1 Functions and Parameters

function add_numbers with a, b
    return a + b
end function

set result to call add_numbers with 5, 10
print result

6.2 Arrays (Lists)

set scores to [10, 20, 30, 40]
set first_score to scores[0]

6.3 Random Numbers

set random_number to random between 1 and 100

7. Error Handling

EZCode makes it easy to handle errors in a simple way by using try-catch blocks.

try
    set result to divide 10 by 0
catch error
    print "Error: Division by zero"
end try

8. Game Engine Integration

8.1 Creating and Moving Objects

create sprite "player" at 100, 100
move sprite "player" left by 5

8.2 Playing Sounds

play sound "explosion.wav"

8.3 Checking Collisions

if sprite "player" collides with sprite "enemy"
    print "Game Over!"
end if

9. Saving and Loading Data

EZCode provides simple methods for saving and loading data, such as game progress or user preferences.

9.1 Saving Data

save data "player_data.txt" as player_name, player_health

9.2 Loading Data

load data "player_data.txt" into player_name, player_health

10. Additional Notes

  • EZCode is constantly evolving, so new features will be added over time.
  • Keep your code simple: EZCode is designed to be beginner-friendly, so avoid overly complex logic or syntax.
  • Join the community: Ask for help on forums, Discord servers, or social media if you get stuck with your EZCode projects.

11. Getting Started

  • Download EZCode interpreter or compiler from the official site.
  • Create a new .ez file and open it with any text editor.
  • Write your first program, such as printing "Hello, World!", and run it!
