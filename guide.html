EZCode Full Guide

Installation

1. Download the latest version from here.

2. Run the installer and follow the steps.

3. Open a terminal and type EZ myprogram.ez to run your first script!

Basic Syntax

Variables

set x to 10

Output

write "Hello, EZCode!"

Input

ask for "What is your name?" as username

Conditions

if x is greater than 5
    write "X is big!"
end if

Loops

repeat 5 times
    write "Hello!"
end repeat

UI Elements

Creating a Button

create button "Click Me" at 100, 200
when button "Click Me" is clicked
    write "Button was clicked!"
end when

Playing Sounds

create sound "beep" from "beep.mp3"
play sound "beep"

Game Development

Creating a Window

create window "My Game" size 800, 600
set background color to "skyblue"

Adding a Player

create sprite "player" from "player.png" at 100, 100

Handling Movement

when key "W" is pressed
    move "player" up 5
end when

Advanced Features

Importing Modules

add module "math"
set y to use sqrt from math with 25
write y

Creating a Server

create server at port 8080
on request
    send response "Hello from EZCode Server!"
end on

Debugging & Errors

  • If you get an "Invalid Syntax" error, check for missing keywords.
  • Ensure all conditions and loops have end if or end repeat.
  • Use write statements to debug variable values.
